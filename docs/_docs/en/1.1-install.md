fijkplayer is a Flutter plugin which warp ijkplayer.  
Add fijkplayer as a dependency in your `pubspec.yaml` file.  
And update the version number to the latest stable version, see the badge bellow.  

```yaml
dependencies:
  fijkplayer: ^0.1.0
```
